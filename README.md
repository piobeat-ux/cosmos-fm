# Cosmos FM - Hotel Media Network

## 🚀 Быстрый старт

```bash
npm install
npm run dev
```

## 🔧 Настройка

1. Скопируйте `.env.example` в `.env`
2. Добавьте ваши credentials из Supabase
3. Выполните SQL из `fix-storage-policies.sql` в Supabase Dashboard

## 📦 Деплой

```bash
npm run build
vercel --prod
```

## 🎨 Особенности

- ✅ PWA поддержка
- ✅ Адаптивный дизайн
- ✅ Оптимизированная загрузка
- ✅ Offline режим
- ✅ Hot reload

## 🛠 Стек

- React 19 + TypeScript
- Vite 5
- Tailwind CSS
- Supabase
- Lucide Icons
